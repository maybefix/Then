# Then

Tauri v2 + React + TypeScript + Vite で作る、縦書き本文執筆向けのエディタ。

リリース時のソフト名は `Then`。Windows 向け Tauri デスクトップアプリとして提供する。
（旧称 `brew`。既存の `.brew/project.json` は `.then/project.json` へ自動移行される。）

## 主な機能

### 執筆
- CodeMirror 6 + 独自の silkdown デコレーションによる縦書きエディタ
- 入力行を画面の一定位置に保つタイプライタースクロール（固定位置を調整可能）
- 縦書き向けの記法
  - 太字 `**…**`（`Ctrl+B`）
  - 圏点 `[…(em,goma)]`（`Ctrl+I`）
  - ルビ `[…(rb,ルビ)]`、青空文庫風 `｜…《ルビ》` も認識
  - 縦中横 `[…(tcy)]`
  - 文章方向 `[(al:start|center|end)]`
  - 見出し 1〜6（`Ctrl+1`〜`Ctrl+6`）／解除（`Ctrl+0`）
- `Ctrl+P` のコマンドパレットから上記操作を実行

### ファイル・ワークスペース
- `.md` / `.txt` ファイルのオープン、保存、別名で保存
- 複数ファイルをタブで同時に編集
- プロジェクトフォルダ配下のサブフォルダと `.md` / `.txt` を辿る階層パンくず
- パンくずメニューからの新規ファイル/フォルダ作成、リネーム、削除、並び替え
- 左サイドバーのファイル表示（ファイルツリー／ナビゲータ）と見出しアウトライン
- このファイル／プロジェクト全体を対象とした全文検索・置換

### 補助ペイン
- スニペット：ダブルクリック／ドラッグ&ドロップで本文に挿入。保存先はフォルダごと／固定プロフィールを選択可能
- アイデア：`#タグ` 付きの短文メモ。本文へドロップ挿入
- プロット：章／セクションのカードで構成管理（連番自動付与、折りたたみ状態を保存）

### 出力・表示
- 複数ファイルを束ねた PDF / Word(DOCX) エクスポート（ページサイズ・余白・段組み・ノンブル等を設定）
- 多数のカラーテーマ（ダーク／ライト）
- Tauri 実行時の Windows OS フォント一覧取得、本文フォント・文字サイズ・行間の設定
- 保存状態表示、文字数表示（空白を含めるか選択可能）、挿入・保存トースト

### 保存
- 本文 Markdown の自動保存・復元、カーソル位置の復元
  - Tauri 実行時はアプリデータフォルダの `app-state.json`
  - ブラウザ実行時は `localStorage`
- スニペット・プロット・並び順はワークスペース内の `.then/project.json`

## 起動

```powershell
npm install
npm run tauri:dev
```

ブラウザで UI 確認だけを行う場合:

```powershell
npm run dev:local
```

## ビルド確認

```powershell
npm run build
npm run tauri:build
```

## ドキュメント

- `docs/USER_MANUAL.md`: 操作マニュアル（エンドユーザー向け）
- `docs/CURRENT_STATE.md`: 現在地点
- `docs/REQUIREMENTS_GAP_LIST.md`: 要件整理と不足リスト
