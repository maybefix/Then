# Then 操作マニュアル

Thenは、小説やシナリオなどの日本語原稿を縦書きで執筆するためのWindowsデスクトップエディタです。

> 対象バージョン: 0.4.6
> 動作環境: Windows x64（Tauriデスクトップアプリ）
> ブラウザ版はUI確認用であり、一部のファイル操作は利用できません。

## Thenとは

Thenは、Markdownまたはテキストファイルを原稿として直接扱う執筆環境です。

- 縦書き／横書き本文エディタ
- タイプライタースクロール
- フォルダ単位の原稿管理
- Idea、Plot、資料、Idea Boardによる構想整理
- プロジェクト全体のチェックポイントと復元
- 複数ファイルを束ねたPDF／DOCX出力

原稿は一般的な `.md` / `.txt` ファイルとして保存されます。Then独自の形式に変換しなくても、ほかのエディタから同じ原稿を開けます。

## 起動する

起動すると「執筆するワークスペースを選んでください。」という起動ポータルが表示されます。

- **フォルダを開く**: 執筆に使うフォルダをワークスペースとして開きます。
- **最近のワークスペース**: 最近開いたフォルダを選んで再開します。
- **履歴を消去**: 最近のワークスペース一覧を消去します。原稿ファイルは削除されません。
- **次回から前回のワークスペースを直接開く**: 次回起動時にポータルを省略します。
- **設定**: ワークスペースを開く前に表示や起動方法を変更できます。

前回のワークスペースが移動・削除されている場合は警告が表示されます。別のフォルダを選び直してください。

## 画面構成

ワークスペースを開くと、画面は次の領域に分かれます。

| 領域 | 役割 |
| --- | --- |
| トップバー | ファイルメニュー、パンくず、画面モード、縦横切替、テーマ、設定。 |
| アクティビティバー | ファイル表示、プロジェクト検索、新規ファイル、新規フォルダ。 |
| 左サイドバー | ファイルツリー／ナビゲータ、見出しアウトライン、検索・置換。 |
| タブ列 | 開いている原稿を切り替えます。 |
| 本文エディタ | 原稿を縦書きまたは横書きで編集します。 |
| 右サイドバー | Idea、Plot、資料。幅の拡張と折りたたみができます。 |
| ステータスバー | 保存状態、文字数、設定に応じてファイルパスを表示します。 |

トップバー中央のモード切替から、次の作業画面を開けます。

- **本文**: 通常の執筆画面
- **キャンバス**: Idea Board
- **エクスポート**: PDF／DOCX出力
- **履歴アイコン**: チェックポイント

キャンバスとエクスポートは、設定により別ウィンドウで開くこともできます。

## ワークスペースとデータ保存

### ワークスペース

「フォルダを開く」で選択したフォルダが1つのワークスペースになります。配下のサブフォルダと `.md` / `.txt` ファイルをThenから管理できます。

サブフォルダを独立したワークスペースとして開くと、そのフォルダ自身に専用のThen管理データが作られ、IdeaやPlotなども別管理になります。

### 保存場所

- 原稿: ワークスペース内の `.md` / `.txt`
- 並び順、Idea、Plotなど: ワークスペース内の `.then/project.json`
- Idea Board: ワークスペース内の `.then/boards`
- 取り込んだプロジェクト資料: ワークスペース内の `.then/references`
- 資料カードの配置: ワークスペース内の `.then/reference-layouts`
- アプリ設定、最近のワークスペース、開いていたタブなど: Tauriのアプリデータフォルダ

`.then` はThenが管理するディレクトリです。通常はファイルを直接編集せず、アプリから操作してください。

### 自動保存

本文は編集後に自動保存されます。ファイルメニューから手動保存することもできます。ステータスバーには「保存済み」「編集中」「保存中」「エラー」などの状態が表示されます。

## ファイルとフォルダ

### ファイルメニュー

トップバー左のメニューボタンから、主に次の操作を行えます。

| 操作 | 内容 |
| --- | --- |
| 新規ファイル | 一時的な新規原稿タブを作ります。 |
| ファイルを開く | 単一の `.md` / `.txt` を開きます。 |
| 保存 | 現在の原稿を保存します。 |
| 別名で保存 | 保存先とファイル名を指定します。 |
| フォルダを開く | ワークスペースを切り替えます。 |
| 前／次のタブ | 編集するタブを切り替えます。 |
| タブを閉じる | 現在のタブを閉じます。 |

### ワークスペース内の操作

ファイルツリー、パンくず、項目メニューから次の操作ができます。

- ファイル／フォルダの新規作成
- 名前変更
- 別フォルダへの移動
- 同一フォルダ内の並び替え
- ファイルの追加
- 削除

削除した項目は即座に完全消去せず、ワークスペース内のThen管理用Trashへ移動します。フォルダを削除する場合は、配下に含まれるファイル数などが確認画面に表示されます。

### パンくず

トップバーのパンくずには、ワークスペースから現在のファイルまでの階層が表示されます。各階層を選ぶと、そのフォルダの内容と操作メニューを開けます。

## タブ

- 複数の原稿を同時に開けます。
- すでに開いているファイルを選ぶと、既存タブへ切り替わります。
- タブを切り替えて戻ったときは、そのタブで見ていた本文位置とカーソル位置を復元します。
- 未保存の変更があるタブは保存状態で区別されます。
- 新規タブには、ファイル作成やワークスペースを開くための案内が表示されます。

## 本文を執筆する

### 縦書きと横書き

既定は縦書きです。トップバーの縦横切替ボタン、または設定の「本文方向」から横書きへ切り替えられます。

### タイプライタースクロール

入力中の行を画面内の一定位置に保つ機能です。

- 設定の「本文」タブでオン／オフを変更できます。
- 「固定位置」で入力行を置く位置を調整できます。
- マウスホイール、クリック、キー入力などの操作を始めると、初期位置調整は中止されます。

### 本文表示幅

設定の「文字表示幅」で、編集領域に対して本文を表示する幅を調整できます。縦書きでは縦方向、横書きでは横方向の表示範囲に反映されます。

### 文字数

ステータスバーに現在の原稿の文字数が表示されます。「文字数に空白を含める」で数え方を変更できます。

## 文字装飾と縦書き記法

選択範囲に対して、ショートカット、右クリックメニュー、コマンドパレットから記法を適用できます。

| 装飾 | 記法 | 操作 |
| --- | --- | --- |
| 太字 | `**本文**` | `Ctrl+B`。同じ操作で解除。 |
| 圏点 | `[本文(em,goma)]` | `Ctrl+I`。同じ操作で解除。 |
| ルビ | `[本文(rb,ルビ)]` | 読みをダイアログで入力。 |
| 縦中横 | `[本文(tcy)]` | 数字などを横並びで表示。 |
| 行揃え | `[(al:start|center|end)]` | 行頭、中央、行末を選択。 |
| 見出し | `#`〜`######` | `Ctrl+1`〜`Ctrl+6`。 |
| 見出し解除 | なし | `Ctrl+0`。 |

青空文庫形式の `｜本文《ルビ》`、漢字直後の `《ルビ》`、`《《圏点》》` も認識します。

ルビ、圏点、縦中横は、改行をまたがない選択範囲に適用してください。「記法をクリア」を使うと、選択範囲または行から独自記法を外せます。

## コマンドパレットとショートカット

`Ctrl+P` でコマンドパレットを開きます。コマンド名を入力して絞り込み、`↑` `↓` で選択、`Enter` で実行、`Esc` で閉じます。

| ショートカット | 操作 |
| --- | --- |
| `Ctrl+P` | コマンドパレット |
| `Ctrl+Alt+I` | Ideaへメモを追加 |
| `Ctrl+B` | 太字 |
| `Ctrl+I` | 圏点 |
| `Ctrl+1`〜`Ctrl+6` | 見出し1〜6 |
| `Ctrl+0` | 見出し解除 |

## 左サイドバー

### アクティビティバー

- **ファイルツリー**: ファイルと見出しを表示します。
- **プロジェクト検索**: 検索・置換画面を表示します。
- **新規ファイル**: 原稿を追加します。
- **新規フォルダ**: 選択中の場所へフォルダを追加します。

### ファイル表示

設定から表示方式を選べます。

- **ファイルツリー**: フォルダ階層をコンパクトに表示します。
- **ナビゲータ**: ファイル冒頭のプレビューを添えて表示します。プレビュー行数も変更できます。

### 見出しアウトライン

- 現在の原稿またはプロジェクト内の見出しを階層表示します。
- 見出しを選ぶと本文の位置へ移動します。
- 現在カーソルがある見出しが強調表示されます。
- 見出しをドラッグすると、セクション単位で本文を移動できます。

### 検索・置換

- 「このファイルを検索」と「プロジェクトで検索」を切り替えられます。
- 検索結果にはファイルと見出しの文脈が表示されます。
- 現在のファイル内の置換と、プロジェクト全体の一括置換に対応します。
- プロジェクト全体を置換する前には確認画面が表示されます。

## Idea

右サイドバーの電球アイコンから開きます。Ideaは「スレッド」と、その中の複数の「断片」で構成されます。

### スレッド一覧

- 検索欄でタイトルや断片を絞り込みます。
- 「すべて」「未使用」「スター」のフィルタを切り替えます。
- 新しいメモを入力し、追加先スレッドを選んで保存できます。
- スレッドを新規作成できます。

### スレッド詳細

- スレッド名の変更、スター、削除
- 断片の追加、編集、並び替え、別スレッドへの移動
- 断片単位または未使用断片一式の本文挿入
- 使用済み状態の切り替え
- 断片またはスレッドをIdea Boardへ送信
- Idea Boardから作った断片は、元の位置を開くことができます。

断片はボタン、ダブルクリック、ドラッグ&ドロップで本文へ挿入できます。`Ctrl+Alt+I` を使うと、右サイドバーを開かずにメモを追加できます。

Ideaの保存先は設定の「スニペット保存先」で「フォルダごと」または「固定プロフィール」を選択できます。

## Plot

右サイドバーの本アイコンから開きます。

- **章**と**セクション**のカードで構成を管理します。
- セクションには `001`、`002` のような番号が自動で付きます。
- 章とセクションはドラッグして並び替えられます。
- セクションを別の章へ移動できます。
- 章を折りたたむと配下のセクションをまとめて隠せます。
- タイトルと本文を直接編集できます。
- Plotから資料を関連付けて開けます。
- 管理画面から全体を一覧できます。

## 資料

右サイドバーの書類アイコンから開きます。

### 対応形式

- テキスト: `.txt`
- Markdown: `.md`
- 画像: `.png` / `.jpg` / `.jpeg` / `.webp`
- PDF: `.pdf`

### 資料の範囲

- **プロジェクト資料**: 現在のワークスペース専用
- **共通資料**: ほかのワークスペースからも利用可能

既存ファイルを取り込むか、新しい `.txt` / `.md` 資料を作成できます。資料は検索・種類フィルタで絞り込み、プロジェクト資料と共通資料の間でコピーまたは移転できます。

資料を開くと、本文上に移動・サイズ変更可能なカードとして表示されます。カードは固定、最前面表示、閉じるなどの操作ができます。テキスト資料はカード内で編集できます。

## Idea Board

トップバーの「キャンバス」、またはIdeaペインのキャンバスボタンから開きます。

- プロジェクト用と共通のBoardを切り替えます。
- 複数のBoardを作成・選択できます。
- テキストカード、資料カード、グループを配置できます。
- Ideaや資料をサイドパネルからクリックまたはドラッグして追加できます。
- カード同士を線、矢印、双方向矢印で接続できます。
- 接続線にはラベルを付けられます。
- カードの複製、削除、サイズ変更、複数選択、グループ移動ができます。
- 選択したテキストカードをIdeaまたはPlotへ送れます。
- Undo／Redo、ズーム、全体表示に対応します。

設定で、カードの既定方向、既定フォント、キャンバスを別ウィンドウで開くかどうかを変更できます。

## チェックポイント

トップバーの履歴アイコンから開きます。チェックポイントは、ワークスペース内の原稿と並び順を保存点として記録する機能です。

### 保存点を作る

「保存点を作成」を選び、名前と必要に応じてメモを入力します。保存点一覧では名前、日時、ファイル数、文字数を確認できます。

### 比較する

- 保存点と現在のプロジェクトを比較
- 2つの保存点を比較
- ファイルごとの差分、追加、削除、移動を確認

### 復元する

- プロジェクト全体を復元
- 選択したファイル／フォルダだけを復元
- フォルダの並び順も復元

現在のファイルと競合する場合は、次から選択します。

- 保存点の内容で置き換える
- 保存点の内容を別ファイルとして復元する
- 競合したファイルをスキップする

復元前には現在の状態が「復元前の退避」として自動保存されます。

## PDFとDOCXのエクスポート

トップバーの「エクスポート」から開きます。

### 出力する原稿

- ワークスペース内のファイルから出力対象を選択します。
- ファイルの順序を変更し、1つの文書として束ねます。
- ファイルごとの開始方法を指定できます。

### レイアウト

- B6、A5、A6、B5、A4、カスタムサイズ
- 天、地、ノド、小口の余白
- 出力フォント
- 段数と段間
- ヘッダー
- フッターとノンブル
- 奇数／偶数ページや章扉での表示制御

プレビューでページを確認し、「PDFを書き出す」または「DOCXを書き出す」を実行します。生成中は進捗が表示され、完了後に出力先を開けます。

設定でエクスポート画面を別ウィンドウにすることもできます。

## 設定

トップバー右の設定ボタンから開きます。

### 外観

- テーマ
- UIフォント
- UI表示サイズ
- Zoneモードと透明度

### 本文

- 本文フォント
- 見出し・太字フォント
- Plot表示フォント
- 縦書き／横書き
- 文字サイズ、行間、文字表示幅
- タイプライタースクロールと固定位置
- 改行記号
- 文字数に空白を含めるか

### ファイル

- ファイルツリー／ナビゲータ
- ナビゲータのプレビュー行数
- ワークスペース切替時のパス表示
- ステータスバーのファイルパス表示
- 起動時に前回のワークスペースを直接開く
- Ideaの保存先（フォルダごと／固定プロフィール）

### キャンバス

- Canvas nodeの既定方向
- Canvas nodeの既定フォント
- キャンバスを別ウィンドウで開く

### エクスポート

- DOCX・PDF出力フォント
- エクスポートを別ウィンドウで開く

テーマはトップバーのテーマボタンから直接選ぶこともできます。

## 困ったときは

### 前回のワークスペースが開かない

フォルダが移動・削除されていないか確認し、起動ポータルから現在の場所を選び直してください。

### 保存状態が「エラー」になる

ファイルやフォルダの書き込み権限、別アプリによるロック、空き容量を確認してください。詳細は画面のエラー表示に示されます。

### ルビ・圏点・縦中横を適用できない

範囲を選択し、改行をまたいでいないことを確認してください。カーソルを置いただけの状態や複数行の選択では適用できません。

### 資料やチェックポイントの操作が使えない

ワークスペースを開いていることと、Tauriデスクトップ版で実行していることを確認してください。

### Windowsがインストーラーを警告する

現在の配布インストーラーはコード署名されていないため、Windows SmartScreenが警告を表示する場合があります。必ず公式の[GitHub Releases](https://github.com/maybefix/Then/releases)から取得してください。
