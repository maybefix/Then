# Then 操作マニュアル

縦書き本文執筆のためのデスクトップエディタ **Then**（リリース名 `brew`）の操作マニュアルです。
小説・シナリオなど、まとまった日本語の本文を書くことを想定しています。

> 対象バージョン: 0.3.2
> 動作環境: Windows（Tauri デスクトップアプリ）。ブラウザでは UI 確認のみ可能です。

## Then とは

Then は、Markdown を下敷きにした縦書き中心の執筆エディタです。主な特徴は次のとおりです。

- **縦書きエディタ**：本文を縦書きで表示・編集できます。
- **タイプライタースクロール**：入力中の行を画面の一定位置に固定して書き続けられます。
- **縦書き向けの記法**：ルビ・圏点・縦中横・文章方向など、日本語組版に必要な装飾を入力できます。
- **フォルダ単位の執筆管理**：プロジェクトフォルダ配下の `.md` / `.txt` を横断して編集・検索できます。
- **アイデア／プロット管理**：右サイドバーに思いつきメモ（アイデア）と構成（プロット）を置けます。
- **組版エクスポート**：複数ファイルを束ねて PDF / Word(DOCX) に書き出せます。

本文の保存は自動で行われ、アプリを閉じても続きから再開できます。

## 画面構成

起動すると、画面は大きく次の領域に分かれます。

```
┌──────────────────────────────────────────────────────────┐
│ ☰ メニュー   Then   パンくず …            設定 履歴       │ ← トップバー
├───────────┬──────────────────────────────┬──────────────┤
│           │                              │              │
│ 左サイド   │        本文エディタ          │  右サイド     │
│ バー       │       （縦書き）             │  バー         │
│（ファイル  │                              │（アイデア／   │
│ /見出し/   │                              │  プロット）   │
│ 検索）     │                              │              │
├───────────┴──────────────────────────────┴──────────────┤
│ 保存状態 ・ 文字数 …                          ステータス  │ ← ステータスバー
└──────────────────────────────────────────────────────────┘
```

| 領域 | 役割 |
| --- | --- |
| トップバー | 左の `☰` がファイルメニュー。中央にパンくず（現在のフォルダ階層）、右に設定・履歴ボタン。 |
| 左サイドバー | ファイル一覧・見出しアウトライン・検索／置換。`◧` ボタンで表示／非表示を切り替え。 |

| 右サイドバー | 「アイデア」「プロット」タブ。畳む／広げるボタンあり。 |
| ステータスバー | 保存状態と文字数を表示。 |

## 起動とデータの保存

### 保存される場所

- **デスクトップアプリ（Tauri）実行時**：アプリデータフォルダ内の `app-state.json` に、開いていたフォルダ・ファイル・設定・カーソル位置などが保存されます。次回起動時に前回の続きから開きます。
- **ブラウザ実行時**：ブラウザの `localStorage` に保存されます（UI 確認用）。

### 自動保存

- 本文の編集内容は自動で保存されます。手動保存（`保存`）も可能です。
- 保存状態はステータスバーに表示されます（保存済み／編集中／保存中／エラー）。

> 大きな変更や機能の検証を始める前に、フォルダを Git 管理している場合はコミットしておくと安全です。

## ファイルとフォルダの操作

操作はトップバー左の `☰` ファイルメニューから行います。

| メニュー項目 | 内容 |
| --- | --- |
| 新規ファイル | 新しい本文ファイル（タブ）を作成します。 |
| ファイルを開く | 単一の `.md` / `.txt` ファイルを開きます。 |
| 保存 | 現在のファイルを保存します。 |
| 別名で保存 | 名前を付けて別ファイルとして保存します。 |
| エクスポート… | PDF / DOCX 書き出し画面を開きます（後述）。 |
| 前のタブ / 次のタブ | 開いているタブを切り替えます。 |
| タブを閉じる | 現在のタブを閉じます。 |
| フォルダを開く | プロジェクトフォルダを開き、配下のファイルを横断管理します。 |

### フォルダ（ワークスペース）

「フォルダを開く」で選んだフォルダがワークスペースになります。

- 配下のサブフォルダと `.md` / `.txt` ファイルが左サイドバーとパンくずに表示されます。
- パンくずの各階層をクリックすると、その階層の項目メニューが開き、**新規ファイル／フォルダ作成・リネーム・削除・並び替え**ができます。
- 並び順は `.then/project.json` に保存されます。既存の `.brew/project.json` は読み込み時に移行されます。

## タブ

複数のファイルをタブで同時に開けます。

- ファイルを開くと既存タブがあればそれをアクティブにし、なければ新しいタブで開きます。
- 編集中（未保存）のタブは保存状態で区別されます。
- タブの切り替え・クローズはファイルメニュー、または本文上部のタブ列から行えます。

新規タブの開始ページでは、新規ファイル作成やフォルダを開くなどの入口が表示されます。

## 本文の執筆

### 縦書きエディタ

本文は縦書きで表示されます。Markdown の見出し（`#`）や引用（`>`）、リストなどがそのまま反映されます。

### タイプライタースクロール

入力中の行を画面の一定位置に保ちながら書き続けられる機能です。

- 設定でオン／オフを切り替えられます（既定はオン）。
- 行の固定位置（画面に対する割合）も設定で調整できます。

### 文字数カウント

ステータスバーに本文の文字数が表示されます。設定で**空白文字を含めるかどうか**を切り替えられます（サロゲートペアは 1 文字として数えます）。

## 文字装飾と縦書き記法

選択範囲に対して、ショートカット・右クリックメニュー・コマンドパレットから装飾を適用できます。

| 装飾 | 記法 | 説明 |
| --- | --- | --- |
| 太字 | `**本文**` | 選択範囲を太字にします（トグル）。 |
| 圏点（傍点） | `[本文(em,goma)]` | 選択範囲に圏点を付けます（トグル）。 |
| ルビ | `[本文(rb,ルビ)]` | 選択範囲に読みを付けます。読みは入力ダイアログで指定します。 |
| 縦中横 | `[本文(tcy)]` | 縦書き中で数字などを横に並べます。 |
| 文章方向 | `[(al:start|center|end)]` | 対象行の文末に行頭／中央／行末揃えの指示を付けます。 |

補足：

- **太字・圏点・見出し**は同じ操作をもう一度行うと解除されます（トグル）。
- ルビ・縦中横・圏点は **単一行の範囲を選択**しているときに使えます（複数行は対象外）。
- 青空文庫風のルビ記法（`｜本文《ルビ》`、漢字直後の `《ルビ》`）や `《《圏点》》` も認識されます。
- 不要になった記法は「記法をクリア」で選択範囲・行からまとめて外せます。
- 見出しはレベル 1〜6 を設定でき、`見出しを解除`（`Ctrl+0`）で本文に戻せます。

## キーボードショートカットとコマンドパレット

### コマンドパレット

`Ctrl + P` でコマンドパレットを開閉します。入力欄でコマンド名を検索し、`↑` `↓` で選択、`Enter` で実行、`Esc` で閉じます。装飾系の操作はここからすべて実行できます。

### 主なショートカット

エディタにフォーカスがある状態で有効です（`Ctrl+P` と `Ctrl+Alt+I` はどこでも有効）。

| ショートカット | 操作 |
| --- | --- |
| `Ctrl + P` | コマンドパレットを開く／閉じる |
| `Ctrl + Alt + I` | Idea にメモを追加 |
| `Ctrl + B` | 太字 |
| `Ctrl + I` | 圏点（傍点） |
| `Ctrl + 1` 〜 `Ctrl + 6` | 見出し 1〜6 |
| `Ctrl + 0` | 見出しを解除 |

> macOS では `Ctrl` の代わりに `⌘`（Command）でも動作します。

## 左サイドバー（ファイル・見出し・検索）

左サイドバーには次の機能があります。トップバー左の `◧`（左サイドバーを表示）で開閉できます。

### ファイル表示

フォルダ配下のファイルを一覧表示します。表示方式は設定で切り替えられます。

- **ファイルツリー**：フォルダ階層をツリーで表示。
- **ナビゲータ**：各ファイルの冒頭プレビューを添えて表示。プレビュー行数は設定で調整（なし〜数行）。

### 見出しアウトライン

本文中の見出しを階層表示します。検索ボックスで見出しを絞り込めます。現在カーソルがある見出しがハイライトされ、クリックでその位置へジャンプできます。アウトライン上でのドラッグにより、見出し単位でセクションを移動・並び替えできます。

### 検索・置換

本文・プロジェクト全体を対象に全文検索できます。

- 検索範囲は **このファイル** と **プロジェクト全体** を切り替えられます。
- 検索結果には見出しの階層（どの見出しの下にあるか）が表示されます。
- 置換も可能です（プロジェクト横断の一括置換に対応）。

## 右サイドバー（アイデア・プロット）

右サイドバーには「アイデア」と「プロット」の 2 つのタブがあります。畳む／広げるボタンで幅や表示を切り替えられます。

### アイデア

思いついたことを短く書き留めるメモ欄です。

- `+` で新規メモを追加し、その場で本文を編集できます。
- 本文中に `#タグ` を書くと自動でタグとして認識され、検索で絞り込めます。
- メモは **ダブルクリックまたはドラッグ&ドロップ**で本文に挿入できます。
- 上下ボタンで並び替え、削除も可能です。

### プロット

章・セクション単位で構成を管理するカードです。

- カードには **章（chapter）** と **セクション（section）** の 2 種類があります。
- セクションには連番（`001`, `002`…）が自動で振られ、章をまたぐとカウントが整理されます。
- 各カードはタイトルと本文を持ち、展開／折りたたみができます。折りたたみ状態は保存されます。
- プロットはワークスペース（フォルダ）ごとに保存されます。

---

## スニペット

よく使う言い回しや定型文を登録して、本文に素早く差し込める機能です。

- スニペットは **ダブルクリック／ドラッグ&ドロップ**でカーソル位置に挿入できます。
- タイトル・本文・カテゴリ・タグを持ち、検索で絞り込めます。
- 並び替え・編集・削除ができます。

### 保存先の選択

設定の「スニペット保存先」で保存場所を選べます。

| 保存先 | 内容 |
| --- | --- |
| フォルダごと | 開いているワークスペースフォルダ内に保存。作品ごとにスニペットを分けたい場合。 |
| 固定プロフィール | アプリ全体で共通のスニペットとして保存。どのフォルダでも同じものを使いたい場合。 |

## エクスポート（PDF / DOCX）

ファイルメニューの「エクスポート…」から、組版された **PDF** または **Word(DOCX)** を書き出せます。プレビューは実際の出力と同じエンジンで描画される WYSIWYG です。

### 出力するファイルの選択

- ワークスペース内のファイルを一覧から選び、出力に含めるかどうかをチェックで指定します。
- 上下ボタンで並び順を変更できます（束ねる順序）。
- 各ファイルの「開始」位置（章扉から始める等）を指定できます。

### レイアウト設定

| 項目 | 内容 |
| --- | --- |
| ページサイズ | B6 / A5 / A6 / B5 / A4 / カスタム（幅・高さを mm 指定） |
| 余白 | 天・地・ノド・小口を mm で指定 |
| 本文フォント | 源ノ明朝（明朝体）／源ノ角ゴシック（ゴシック体） |
| 段組み | 段数・段間（mm） |
| ヘッダー | 章扉では非表示／先頭ページでは非表示／奇数・偶数で出し分け |
| フッター（ノンブル） | ページ番号の表示、内容（なし／ページ番号／作品名／任意テキスト） |

### 書き出し

画面下部の「PDFを書き出す」「DOCXを書き出す」で実行します。生成中は進捗（現在ページ／総ページ・パーセント）が表示されます。出力形式は画面上部の `PDF` / `DOCX` 切り替えでも選べます。

## 設定

トップバー右の「設定」ボタンから設定モーダルを開きます。

| 設定項目 | 内容 |
| --- | --- |
| テーマ | 多数のカラーテーマから選択（ダーク／ライト各種）。テーマ選択画面から切り替えます。 |
| 本文フォント | 本文の表示フォント。Windows ではインストール済みフォント一覧から選べます。 |
| 文字サイズ | 本文の文字サイズ。 |
| 行間 | 本文の行の高さ。 |
| タイプライタースクロール | オン／オフと固定位置（画面に対する %）。 |
| 改行記号を表示 | 改行位置に記号を表示するかどうか。 |
| 文字数に空白を含める | 文字数カウントに空白を含めるか。 |
| ファイル表示方式 | 左サイドバーのファイル表示（ファイルツリー／ナビゲータ）。 |
| ナビゲータのプレビュー行数 | ナビゲータ表示時の冒頭プレビュー行数。 |
| スニペット保存先 | フォルダごと／固定プロフィール。 |

## 困ったときは

- **前回の続きが開かない**：ワークスペースフォルダやファイルが移動・削除されていないか確認してください。見つからない場合は一時ファイルとして起動します。
- **保存状態が「エラー」になる**：ファイルの書き込み権限や、外部からの編集・ロックを確認してください。エラーの詳細は画面に表示されます。
- **カーソル位置が先頭に戻る**：本文がアプリ外で編集されると保存時の長さと食い違い、復元できず先頭に戻ります（仕様）。
- **ルビ・圏点・縦中横が使えない**：選択範囲に改行が含まれていると無効になります。範囲をきちんと選択したうえで（カーソルを置いただけの空選択は不可）、改行をまたがない 1 行内の範囲を選んでください。なお、長い段落が画面上で折り返されているだけなら改行は含まれないため有効です。
