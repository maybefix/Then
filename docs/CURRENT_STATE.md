# 現在地点: brew

作成日: 2026-06-02

## 概要

`brew` は Tauri v2 + React + TypeScript + Vite + Milkdown で作る、Markdown本文執筆向けのスニペットエディタ。

現在はプロトタイプ UI から、Markdownファイル保存、設定、スニペットCRUDを持つ実用最小版へ進めている。

## 実装済み

- Tauri v2 のデスクトップアプリ構成
- リリース時のソフト名、ウィンドウタイトル、バンドル名を `brew` に統一
- React + TypeScript + Vite のフロントエンド構成
- Milkdown / ProseMirror による Markdown WYSIWYG 編集
- Markdown serializer による本文 Markdown 文字列の取得
- `.md` ファイルを開く、保存、名前を付けて保存
- フォルダを開き、配下のサブフォルダと `.md` ファイルを再帰的に一覧化するプロジェクト操作
- パンくずを `ルート > サブフォルダ > ファイル` の実データ構造として表示
- パンくずのフォルダドロップダウンから子フォルダ/Markdownファイルを辿る操作
- パンくずメニューからの新規ファイル作成、新規フォルダ作成
- ファイル/フォルダのリネーム、削除
- パンくずメニュー内の上下ボタンによる同一フォルダ内の手動並び替え
- プロジェクトルートの `.brew/project.json` による並び順保存
- 上部のファイル操作をハンバーガーメニューへ整理
- Tauri command によるアプリ状態の保存・復元
- ブラウザ実行時の `localStorage` フォールバック
- 自動保存と保存状態表示
- 上部パンくずリスト
- 右側スニペットパネル
- スニペットの検索、追加、編集、削除、並び替え
- スニペット追加・編集は `+` ボタンまたはカード編集ボタンから開くモーダルで行う
- スニペットカードの title / preview / category / tags / 文字数表示
- ダブルクリックによる現在カーソル位置へのスニペット挿入
- ドラッグアンドドロップによる任意位置へのスニペット挿入
- ドラッグ中の横線インジケータ
- 設定は右上ボタンから開くモーダルで行う
- Tauri 実行時に Windows の OS フォント一覧を取得し、本文/UIフォント候補として使う
- フォント、文字サイズ、行間の設定
- タイプライタースクロールのオン/オフと固定位置設定
- 画面下部ステータスバー
- 本文 `textContent` に基づく文字数表示
- 挿入・保存完了トースト
- リリース実行時の Windows コンソール表示抑制
- UI確認用 `npm run dev:local`

## 確認済み

- `npm run build`
- `npm run tauri:build`

どちらも成功。`tauri:build` では以下が生成される。

- `src-tauri/target/release/brew.exe`
- `src-tauri/target/release/bundle/msi/brew_0.1.0_x64_en-US.msi`
- `src-tauri/target/release/bundle/nsis/brew_0.1.0_x64-setup.exe`

## 現在の制約

- フォルダ削除は安全のため空フォルダのみ対応
- 並び替えは上下ボタン操作のみ。ドラッグによる並び替えは未実装
- リネーム時の `.brew/project.json` 内 order 名更新、不要 order エントリ掃除は未実装
- バージョン履歴、最近開いたファイル、ネイティブメニューは未実装
- ブラウザ実行時は OS フォント一覧取得に制限があるため、フォールバック候補を表示する
- スニペットは本文挿入後に元スニペットとの関連を保持しない
- 自動テストは未整備
