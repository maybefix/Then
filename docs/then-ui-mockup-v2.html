# 第一章
## 目覚め

静寂のなかで、目はゆっくりと開いた。
天井の木目が、まだ焦点を結ばない。
視界の端に、薄い朝の光が揺れている。
手足は思うように動かなかった。
ただ、窓の向こうで鳥が一度だけ鳴いた。
それが現実だと、私はその音で知った。

  記憶のどこかに、置いてきた痛みがある。
木のきしむ音が、やけに大きく響く。
何かを思い出そうとすると、
胸のあたりに鈍い熱がかかった。
けれど、進まなければならない気がした。
理由はわからない。ただ、そう感じた。
