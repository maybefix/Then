# Vivliostyle Viewer

Then bundles **Vivliostyle Viewer** for PDF layout (CSS Paged Media).
The Viewer distribution is used unmodified and runs as a separate program in
its own hidden webview. Then does not import or link the Vivliostyle source
code into its React / TypeScript bundle.

- Upstream repository: https://github.com/vivliostyle/vivliostyle.js
