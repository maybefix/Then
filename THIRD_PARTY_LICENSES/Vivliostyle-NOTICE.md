# Vivliostyle Viewer

This application bundles the **Vivliostyle Viewer** to lay out PDF exports
(CSS Paged Media). It is used **unmodified**, as a separate program loaded into
its own webview; the application's own source code does not import or link the
Vivliostyle source into its bundle.

- Component: `@vivliostyle/viewer`
- Version: 2.43.3
- License: **AGPL-3.0** (dual-licensed; a commercial license is available from

- Bundled at: `public/vendor/vivliostyle-viewer/`
- Upstream source: https://github.com/vivliostyle/vivliostyle.js (tag for v2.43.3)
- Project site: https://vivliostyle.org/
